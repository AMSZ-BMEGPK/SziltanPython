Python programozási nyelv¶

A Python egy open-source (OS), interpretlt, ltalnos cl programozsi nyelv (vagy script-nyelv).

Tulajdonsgai:

  • Objektum orientlt
  • Interpretlt
    • Nem szksges fordtani (mint a pl a C++-t), elg csak berni a parancsot, s mr futtathat is a kd
    • Alkalmas teszi szmtsok gyors-prototipizlsra
    • Cserbe lass
  • Open-source:
    • Ingyenes
    • Folyamatosan karban tartott
    • Szles krben felhasznlt iparban s akadmiban is
    • Nagy "Community", sok segdlettel, frummal (pl.: stackoverflow)
  • Modulris:
    • Rengetek feladatra ltezik "package" (pl.: numerikus szmtsokra numpy/scipy, szimbolikus szmtsokra sympy, tblzatfjl-kezelsre CSV)
    • Csak azt kell behvni, amire szksgnk van
    • Ismerni kell a package ekoszisztmt, mik lteznek, mi mire j, stb...
  • Sok IDE (Integrated Development Environment) ltezik:
    • Alapveten shell (terminl) alap
    • Notebook: jupyter notebook, jupyter lab
    • Szvegszerkeszt: Spyder, VS Code (ingyenes/open source - ezek tartalmaznak Debugger-t is)
    • Fizets szvegszerkeszk (lista nem teljes): Visual Studio, PyCharm, stb...

Jupyter notebook működése (+ Python kernel):¶

Legfontosabb tudnivalk:

  • Csak egy front-end, ami kommunikl egy kernel-lel (ez a kernel menben vlaszthat).
  • Kt md ltezik:
    • Command mode (cellamveleteket lehet vgezni)
    • Edit mode (szvegbevitel cellba)
  • Command mode (ESC billenty lenyomsval rhet el, kk csk a cella kijellse esetn):
    • Notebook mentse: s
    • Cellk hozzadsa: b az aktulis cella al, a az aktulis cella fl
    • Cella trlse: ktszer egyms utn a d billenty lenyomsa
    • Cella trlsnek visszavonsa: z
    • Cella msols: c, kivgs: x, beilleszts az aktulis cella al: v
    • Szmozs bekapcsolsa a cella soraira: l (kis L), vagy Shift + l az sszes cellra
    • Cellamdok: futtatand kd: y, nyers kd (nem futtathat): r, markdown (formzott szveg): m
  • Edit mode (Enter billenyt lenyomsval rhet el, zld szn):
    • Sor "kikommentelse"/"visszalltsa": Ctrl + /
    • Tbb kurzor lehelyezse: Ctrl + Bal egrgomb
    • Tglalap-szer kijells (rectangular selection): Alt + Bal egrgomb "hzsa" (dragging)
  • Kzs
    • Cella futtatsa, majd cellalptets: Shift + Enter (ez ltrehoz egy j cellt, ha nincs hova lpnie)
    • Cella futtatsa cellalptets nlkl: Ctrl + Enter

Saját függvények ¶

Szerkezet:

def function(*arguments):
    instruction1
    instruction2
    ...
    return result

A fggvny al tartoz utastsokat tabultoros behzssal (indent) kell berni (nincs {} zrjel, vagy end). A fggvny neve utn jnnek az argumentumok majd kettsponttal : jelezzk, hogy hol kezddik a fggvny.

Külső függvénykönyvtárak: ¶

A mrnki gyakorlatban elfordul alapszmtsokhoz 2 f csomag importlsa ajnlott:

  • sympy: szimbolikus szmtsokhoz
  • scipy/numpy: numerikus szmtsokhoz (pl mtrix algebra)
  • matplotlib.pyplot: Fggvnyek brzolsa A module-ok hvsa az albbi mdokon trtnhet
  • import modulename : importlja az adott module-t. Ekkor a module-ban lv fggvnyek hvsa a module.functionname(arguments) segtsgvel trtnik (Az 1_Alapok.ipynb-ben lthattunk erre pldt a math module esetben).
  • import modulename as alias : hasonl az elzhz, de megvlasztjuk, hogy milyen alias-knt hivatkozunk a module-unkra
  • from modulename import function1, function2, ... : csak bizonyos fggvnyek importlsa (nem szksges a module-ra hivatkozni a fggvnyek hvsa sorn)
  • from modulename import * : a module sszes fggvnynek importlsa (nem szksges a module-ra hivatkozni a fggvnyek hvsa sorn)
